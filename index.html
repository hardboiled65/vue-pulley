vue-pulley

vue-pulley is the Vue directive for custom scrollbar that mimics macOS scrollbar.

Demo

To use vue-pulley, you should make 2 elements first.

<!-- el1 is the main element that v-pulley directive inserted. -->
<div class="el1" v-pulley>
  < !-- el2, contents area you want to scroll. -->
  <div class="el2">
    <p>Your main contents here.</p>
  </div>
</div>
